# 👟 Online Shoes Shopping Website

## 📌 Description
A full-stack e-commerce web application that allows users to browse products, add items to the cart, and place orders online.

## 🚀 Features
- User login and registration
- Product listing and details
- Shopping cart functionality
- Admin panel for managing products and users

## 🛠️ Tech Stack
Frontend: HTML, CSS, Bootstrap  
Backend: PHP  
Database: MySQL  

## 🎯 Purpose
To build an online shopping platform with complete product and user management.
